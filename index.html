Please note the following things:

  • This app is very much beta, and will have some rough edges.
  • This app only works with MP3 and OGG/Vorbis files.
  • You can only edit the metadata the music app cares about. You can keep other metadata, though.
  • The track and disc number should be pure numeric, optionally followed by a slash and the total number. You won't be prevented from entering anything else, though.
  • You can add cover images to OGG files, but only recent versions of Firefox OS will read them (probably, I didn't actually test this). Also note that large images will break the file. The threshold is somewhere around 100 KB. Cover files in MP3 should work with much less problems.
  • You can trim MP3 files (remove parts at the start and the end). Times should be entered in the form min:sec. You can use decimals to specify milliseconds. Leave the fields empty if you want to trim only from one side or not at all. Note that this might not always work as expected and might confuse the time slider. You should consider this feature as very experimental, you might want to use a different tool like wincreator.com instead.
  • The original file is kept unchanged, instead a new copy will be stored in your Download folder. Please make sure that this new file works as expected before replacing the original.
  • This app is free software, its source can be obtained on Github. There you also find the exact conditions under which you are allowed to reuse this software. This software is provided 'as-is', without any express or implied warranty.
